FILM HUNT

HTML | CSS | UX DESIGN | UI DESIGN

I and a friend always felt frustrated when trying to find a good movie to watch at home that night. We liked to use IMDB because we felt that it's ratings were the most accurate, however, using it's advanced search was not easy. Amoung the dozens of options given to us, we only ever wanted to use a couple of search options: the top rated movies, within a certain genre and within a certain time period. Because of this, we decided to create filmhunt.co. Created within 2 weeks, Film Hunt allows you to find the top movies from a genre and a time period using IMDB's API.

The premiss is simple and so too I wanted to have a simple user experience. All the user has to do to begin their search is to fill in the options that are wrapped around a straight forward sentence; "I want the best x-genre movies from x-years" and then press the big button that reads "SHOW ME". From here the user is given a list of these movies decending from top rated to lowest rated. The bare minimum for information that we felt was necessary for picking a movie to watch is shown with more important information stading out more than the rest.

For the visual style I wanted to something modern using a dark pallet, reminisent of a dark theatre. The top background image is shown subtly and changes depending on what genre is chosen, given each genre a different feel.
