FILM HUNT

HTML | CSS | UX DESIGN | UI DESIGN | BRANDING

Nelson Shaw and I found we always felt frustrated when trying to find a good movie to watch at home. We enjoyed using IMDB as it's ratings were the most accurate however, using it's advanced search was not easy. Out of the dozens of search options available, we only ever wanted to use a couple: the top rated movies, within a certain genre and within a certain time period. Our solution - to develop and design Film Hunt, a website which allows you to find the top movies from a genre and a time period using IMDB's API.

We wanted Film Hunt to have a simple user experience to match its simple premise. All the user has to do to begin their search is fill in the blanks in one simple sentence - "I want the best xxx-genre movies from xxx-years". They then press the big "SHOW ME" button and from here the user is presented with a descending list of movies ranked from top-rated to lowest-rated.

For the visual style I chose a modern aesthetic, using a dark palette reminiscent of a movie theatre. The top background image is shown subtly and changes depending on what genre is chosen providing each genre with a different feel.
